# PRODIGY_DS_04
Task 4: Social Media Sentiment Analysis and Visualization

Objective

Analyze and visualize sentiment patterns in social media data to understand public opinion and attitudes toward specific topics or brands.

Dataset

Twitter Entity Sentiment Analysis Dataset

Tools Used
	•	Python
	•	Pandas
	•	Matplotlib
	•	Jupyter Notebook

Tasks Performed
	•	Loaded and explored the social media dataset.
	•	Cleaned and prepared the data for analysis.
	•	Analyzed positive, negative, and neutral sentiments.
	•	Created visualizations to compare sentiment distribution across different entities.
	•	Identified public opinion trends and sentiment patterns.

Key Insights
	•	Compared sentiment distribution among different brands and topics.
	•	Identified entities with the highest positive and negative sentiment.
	•	Visualized overall sentiment patterns using charts.
	•	Gained valuable insights into public opinion from social media data.

Outcome

Successfully analyzed social media sentiment and created visualizations to understand public attitudes toward different brands and topics.
